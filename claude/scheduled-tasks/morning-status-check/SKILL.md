毎朝の状況確認タスクです。以下の手順で実行してください。

## 目的
BacklogとGoogleカレンダーを確認し、今日やることをSlackに通知する。

## 手順

### 1. Backlogタスク確認
- Backlog MCPツールを使って、未完了タスクを最大10件取得する
- スペース: tunapackag.backlog.com
- 優先度・期限・ステータスを取得する

### 2. Googleカレンダー確認
- 今日の日付の予定を全件取得する（timeZone: Asia/Tokyo）
- 時間・件名・場所（あれば）を取得する

### 3. Slackに通知
- Slack MCPツールを使って、以下の形式でメッセージを送信する
- 送信先チャンネル: #general（なければDMでも可）

メッセージ形式：
```
【今日の状況 {日付}】

📋 Backlog未完了タスク（上位5件）
・{タスク名}（優先度: {高/中/低}）
・...

📅 今日の予定
・{時間} {件名}
・...

今日もよろしくお願いします！
```

## 注意
- 日本語で通知すること
- タスクや予定が0件の場合もその旨を通知すること
